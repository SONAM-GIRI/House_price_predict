About This Project

House Price Predictor is a machine learning web app built using Flask and scikit-learn. It allows users to estimate the market value of a house based on factors like size, number of bedrooms, age, and location score.

This project demonstrates:

  • Data handling and preprocessing with Pandas
  • Model training using Linear Regression from scikit-learn
  • Building a web interface using Flask and HTML/CSS
  • User interaction with forms and result rendering
  • Deploy-ready design with modular routing

This project was developed by Sonam Giri (Sonam-Giri) to showcase full-stack AI integration in a simple and intuitive way.
